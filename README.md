## Keyboard Controls
### Link Controls
#### Link Move Up
- ```W```
- ```↑```
#### Link Move Down
- ```S```
- ```↓```
#### Link Move Left
- ```A```
- ```←```
#### Link Move Right
- ```D```
- ```→```
#### Link Attack with Sword
- ```N```
#### Link Attack with A Weapon
- ```X```
#### Link Attack with B Weapon
- ```Z```
### Game Controls
#### Pause Game
- ```Left Shift```
#### Quit Game
- ```Q```
#### Reset Level
- ```R```
#### Open HUD
- ```Tab```
#### Close HUD
- ```Esc```
#### Item Selection
- Cycle Right: ```→```
- Cycle Left: ```←```
- Select Item: ```Enter```
### Start Screen
#### Select Easy Mode
- ```↑```
#### Select Hard Mode
- ```↓```
#### Start Game
- ```Enter```
